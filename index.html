Fourth Project

Halloween Tic Tac Toe Game

This HTML and JavaScript code creates a Halloween-themed Tic Tac Toe game. The page features a stylish, responsive layout built with Bootstrap and includes a background image and spooky design elements. The game logic is implemented in jQuery, where a 3x3 grid is dynamically generated and used for players to take turns marking cells with either "X" or "O". The game checks for a winner or a tie after each move and prompts the user accordingly. If a player wins or the game ends in a tie, the board resets for a new round. The design is enhanced with a Halloween theme, including a pumpkin image and eerie styling.
